Exercício 5. O que acontece com o if do exercício anterior?

Ao criarmos uma referência "user" para um novo objeto do tipo conta. Estaremos salvando na memória uma posição para guardar os dados referentes a user. 

Conta user = new Conta(); // Reserva um espaço na memória para essa referência (user)

Então, quando dizemos: 

user1 = user ;  // user1 passa a olhar para o mesmo objeto que user. 

Ao usarmos o sinal "=" estamos copiando o valor dela. Nesse caso, a referência dela na memória.  Por mais que user e user1 sejam diferentes em seus atributos. Eles retornarão iguais.
